| 需要进行的功能有：游客注册登录，顾客购物，管理员更改商品信息，顾客退货。 游客进入系统后，需要进行登录或注册操作，系统从而确定用户的身份，来让不同用户进行不同的业务。 顾客为该系统的主要使用者，顾客进行的业务主要有购物和退货两个。顾客购物的流程为：浏览商品，选择需要购买的商品，然后下单，收取快递。顾客退货的流程为：申请退货，返还商品，退货成功。 管理员需要维护系统，保证系统正常工作。管理员可以更改商品的信息，除此之外，管理员需要依据订单进行发货。 |                                                              |              |           |              |              |          |
